# 環境省 クマ生息域データ

## ファイル

- `bear_combined.csv` — 5km メッシュごとのクマ生息記録サマリー
  - 17,069 行（ヘッダー含む、データ 17,068 行）
  - うち非ゼロは 9,427 行（`public/data/mesh.json` に採用）

## CSV スキーマ

| カラム | 型 | 意味 |
|---|---|---|
| `mesh_code` | string (8 桁 JIS) | 5km メッシュコード（先頭 2 桁 = 緯度区画、次 2 桁 = 経度区画…） |
| `second` | int | 第2回 自然環境保全基礎調査（1978 年）の記録数 |
| `sixth` | int | 第6回 調査（2003 年）の記録数 |
| `latest` | int | 最新回 調査（第7回 2018 年想定）の記録数 |
| `latest_single` | int | 直近記録が単発目撃の場合のフラグ（減点要素） |

## 出典

- 環境省 生物多様性センター「自然環境保全基礎調査（哺乳類分布調査）」
  - https://www.biodic.go.jp/kiso/fnd_list_h.html

## 加工履歴

- **一次データの出処**: 環境省の公開データを前プロジェクト
  （Flutter 版くまもりマップ）の段階で集約・整形したもの
  - 回次別の記録を 1 メッシュ 1 行に畳み込み済み
- **最終更新**: 2025-08-18（前プロジェクトでの出力日）
- **verifiedAt**: 未検証（環境省の最新公開データとの突き合わせは未実施）

## 利用上の注意

- このファイルはスコア計算の「履歴（生息域）」因子の基礎データ
- 5km メッシュ粒度のため、メッシュ内の特定地点のリスクを
  正確には表現しない
- 調査年が古いため（最新でも 2018 年想定）、
  直近の出没傾向は Sharp9110 アダプタ（`/api/kuma`）で補完する
- 将来的には一次データからのビルドパイプライン整備が必要
